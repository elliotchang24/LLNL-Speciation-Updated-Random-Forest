To execute the LSURF_v1 Jupyter notebook:

1) Download all items. Most importantly:
   a) phreeqc executable
   b) LLNL_NEA.txt database
   c) LSURF_v1 notebook
   d) sc.subset csv data file

2) Save the files in a single folder. Drag this folder onto Desktop to ensure Permissions are enabled for this file access.
We will be calling on this folder, which contains the phreeqc executable and stores L-SURF outputs.

3) Open LSURF_v1 notebook and modify paths as defined in the "Preparing Paths" section of the code.
Notably, modify the 'phreeqc_dir' -- this is where your phreeqc file is located (on the Desktop).

4) Ensure sklearn version has been updated to the latest, current version for your computer.

5) Click 'Run All Cells'.
